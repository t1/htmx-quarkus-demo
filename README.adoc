A demo for https://htmx.org/docs/[htmx] using https://github.com/t1/bulma-java[bulma-java].

This is a https://quarkus.io[Quarkus] app; start it with:

[source,bash]
----
mvn quarkus:dev
----

Then press `w` in the cli to open the web page.
